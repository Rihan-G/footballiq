# ⚽ FootballIQ

A FotMob-style live football week view powered by Claude AI + web search.

- Full week of fixtures across all major leagues
- Live scores, upcoming kick-offs, finished results  
- Click any match for AI-powered stats, xG, win probabilities & prediction
- Built with React + Vite, deployed to GitHub Pages

## Live Demo

https://rihan-g.github.io/footballiq

## Deploy

```bash
npm install
npm run deploy
```
